# 芸芸偕行 (前端) - yunyun-frontend


## 项目介绍

芸芸偕行，乃以标签和队伍的社交平台，志在协助用户轻松寻觅志趣相投之同道伙伴，联袂出发，共同踏上旅程之路。

此为该项目的前端部分, 后端部分请移步 [yunyun-backend](https://github.com/defen2001/yunyun-backend)

## 项目技术栈

- Vue
- Vant
- TypeScript
- Vite
- VueRouter
- Pinia
- Axios

## 项目结构

```
src                 源代码
├─api               ├─API 接口
├─components        ├─公共组件
├─layouts           ├─布局组件
├─models            ├─数据模型
├─pages             ├─页面
│  ├─home           │  ├─主页
│  ├─mine           │  ├─我的
│  │  └─account     │  │  └─账号
│  └─team           │  └─队伍
├─routers           ├─路由定义
├─store             ├─状态管理
└─utils             └─工具类
```

## 快速启动
```
# 克隆到本地
git clone https://github.com/defen2001/yunyun-frontend.git

# 进入文件夹
cd yunyun-frontend

# 安装依赖
npm install 或 yarn(推荐)

# 开启本地服务器localhost:80
vite
# 发布环境
```
